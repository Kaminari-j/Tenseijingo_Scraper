# Tenseijingo_Scraper  

## 事前準備  
### パッケージをインストール
  ```bash
  pip install -r requirements.txt
  ```
### ユーザー情報を設定  
tenseijingoscraper ディレクトリ配下の userinfo.py にユーザーIDとパスワードを記入
  ```
  id = ''
  password = ''
  ```
